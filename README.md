**Gemini AI Assistant** "gemini-quick-assist" (Windhawk Mod)

This WindHawk mod integrates Google's Gemini API models directly into the Windows 11 workflow, allowing users to process selected text using a keyboard shortcut.

Key Features:
  Trigger: Press Ctrl+Shift+G to send selected text to Gemini.
  Modern UI: Displays responses in a custom, resizable dark-mode window with Rich Text/Markdown support (headers, lists, code blocks).
  Functionality: Includes a "Fix Grammar" mode for instant text replacement and support for custom user-defined prompts.
  UX Improvements: Features an animated loading indicator, keyboard shortcuts (Esc to close, Ctrl+A to select), and refined visual styling.

Configuration:
  Required: Users must provide their own Gemini API Key from Google AI Studio.
  Customizable: Users can change the AI model (default: gemini-flash-latest), modify the system prompt, or add custom menu options via the settings tab.
